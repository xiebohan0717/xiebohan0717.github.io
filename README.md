# 学术个人网页

这是一个简单的学术个人网页项目，使用HTML和CSS构建，展示个人信息，并准备部署到GitHub Pages。

## 部署到GitHub Pages

1. 在GitHub上创建一个新的公共仓库。
2. 将此项目的文件推送到仓库的主分支。
3. 在仓库的设置页面，导航到"Pages"部分。
4. 选择"Deploy from a branch"，然后选择主分支作为源。
5. 保存设置后，GitHub Pages将自动构建和部署您的网站。
6. 网站将在 `https://<您的用户名>.github.io/<仓库名>/` 上可用。

## 本地预览

打开 `index.html` 文件在浏览器中查看网页。

## 自定义

- 编辑 `index.html` 添加更多内容，如研究项目、出版物等。
- 修改 `style.css` 调整样式。
- 添加图片或其他资产到项目中。
